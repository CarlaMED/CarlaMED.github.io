Data Analytics -Python

Descriptive analytics was used to describe the data by summarizing its main characteristics. This included finding the mean, median, mode, and standard deviation of the data. Hypotheses were defined and tested for the dataset. This involved using statistical tests to determine if there was a significant relationship between two or more variables. Classification techniques were used to classify data points into one of two or more categories. Association rules techniques were used to find relationships between two or more variables. For each technique chosen for the prediction models, an analysis of its effectiveness in relation to this dataset was provided.
